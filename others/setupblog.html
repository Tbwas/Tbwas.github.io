# 博客搭建

以前，用hexo搭了个博客，简单快捷，部署也很方便。虽然官方提供的主题很多，但一直没有自己喜欢的样式，个人比较喜欢简约而又不单调的，直到遇见了vuepress，便认定它了。

步骤如下：

1. 安装vuepress

npm install -g vuepress

2. 在本地新建一个博客目录，名称不限

mkdir xxxblog

3. 进入到该目录下，运行如下命令，

npm init -y

之后，博客目录下便会生成一个package.json文件，可以在该文件中配置一些命令，例如：

"scripts": {
    "dev": "vuepress dev docs", // 运行
    "build": "vuepress build docs", // 打包
    "deploy": "bash deploy.sh",  // 部署
  }

deploy.sh配置参考官网，同时需要在当前目录下创建一个名为docs目录，用作博客的根目录。

4. 在docs目录下，再建一个.vuepress目录以及一个README.md文件。.vuepress目录用来存放配置文件和一些图标资源等，README.md文件用于展示博客首页。

5. 在.vuepress目录下，需要建一个config.js配置文件，还需要再建个public目录，用于存放图标资源等。

上述步骤完成后，你的博客目录结构应该是这样：

├── docs
│   ├── .vuepress
│   │   ├── config.js
│   │   └── public
│   └── README.md
└── package.json

config.js具体如何配置，直接看官方文档即可。

配置完成后，接下来就可以写博客了。关于部署，参考官网即可。

这里有两个点需要注意下：

  1. 若在根目录(docs)下，创建了其他目录，每个目录下都要有个README.md文件，不然无法显示；

  2. 像网页图标favicon以及首页展示的一些图片，都要放在public文件下，不然无法加载。
